# 该项目是关系抽取加实体抽取

启动 app.py 文件，实现离线部分的关系抽取，实体抽取，送入 neo4j 数据库当中。

启动 query.py 文件，实现在线部分的关系抽取，实体抽取，进入 neo4j 数据库查询对应的实体和对应的关系。
